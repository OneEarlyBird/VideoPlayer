项目说明
------

* 主题：基于迅雷APlayer媒体播放引擎的视频播放器，采用DMSkin控件集美化
* 功能：打开视频播放、暂停、停止、调整音量、截图、全屏播放、隐藏列表、设置截图路径、批量导入视频等
* 知识：C#基础知识、APlayer媒体播放引擎API、DMSin的集成使用
* 支持：如果觉得还不错，点个 Star 就是给我开源代码最大的支持 O(∩_∩)O...

* 注意：如果需要本源码的文档，请发邮件联系我！
